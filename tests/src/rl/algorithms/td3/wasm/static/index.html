BackpropTools: The Fastest Deep Reinforcement Learning Library for Continuous Control That Runs Everywhere

In the previous example the Pendulum-v1 swingup is trained directly in your browser using the TD3 deep reinforcement learning algorithm. BackpropTools is a pure C++ header-only, dependency free deep supervised- and reinforcmeent learning library that runs on a broad variety of devices (as e.g. your browser). To run it in a browser the C++ code is compiled to WASM which can be called from JavaScript. For more information read the paper [link] and/or the source code [link].
